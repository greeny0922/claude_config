# Code Explainer

Analyze and explain code in an easy-to-understand manner.

## Target
$ARGUMENTS

## Analysis Scope

### 1. Overview
- One sentence summary of what this code does
- Main purpose and role

### 2. Structure Analysis
- List of main functions/classes
- Dependencies and import relationships
- Data flow

### 3. Core Logic
- Algorithm explanation (if applicable)
- Important business logic
- Detailed explanation of complex parts

### 4. Usage
- How to call/use this code
- Required inputs and expected outputs
- Example code snippets

### 5. Caveats
- Known limitations
- Edge cases
- Performance considerations

## Output Format
- Use markdown formatting
- Use language tags in code blocks
- Use Mermaid syntax for diagrams if needed

## Language
**Always respond in Korean when explaining to the user.**
